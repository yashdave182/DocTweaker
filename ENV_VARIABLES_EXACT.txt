================================================================================
HOW TO ADD IN VERCEL DASHBOARD
================================================================================

Step 1: Go to Vercel project settings
   https://vercel.com/dashboard → Your Project → Settings

Step 2: Click "Environment Variables" in left sidebar

Step 3: Add first variable
   Name:        UNSTRACT_API_KEY
   Value:       [paste your actual key here]
   Environment: ☑ Production ☑ Preview ☑ Development
   Click "Save"

Step 4: Add second variable
   Name:        GEMINI_API_KEY
   Value:       [paste your actual key here]
   Environment: ☑ Production ☑ Preview ☑ Development
   Click "Save"

Step 5: Redeploy
   Deployments → Latest → "..." menu → "Redeploy"


================================================================================
COMMON MISTAKES TO AVOID
================================================================================

❌ WRONG: unstract-api-key (lowercase with dashes)
✅ CORRECT: UNSTRACT_API_KEY (uppercase with underscores)

❌ WRONG: UnstractApiKey (camelCase)
✅ CORRECT: UNSTRACT_API_KEY

❌ WRONG: UNSTRACT API KEY (spaces)
✅ CORRECT: UNSTRACT_API_KEY

❌ WRONG: @unstract-api-key (with @ prefix - that's for secrets)
✅ CORRECT: UNSTRACT_API_KEY

❌ WRONG: Adding in vercel.json file
✅ CORRECT: Adding in Vercel Dashboard only


================================================================================
VERIFICATION
================================================================================

After adding environment variables, test by visiting:
https://your-app.vercel.app/api/health

Expected response:
{
  "status": "healthy",
  "unstract_configured": true,  ← Should be true
  "gemini_configured": true,    ← Should be true
  "environment": "vercel"
}

If they show false:
1. Check variable names are EXACTLY as shown above
2. Check you selected all three environments (Production, Preview, Development)
3. Redeploy the application
